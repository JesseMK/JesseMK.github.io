##eval
(PHP 4, PHP 5)
eval ― 把字符串作为PHP代码执行
###说明
```
mixed eval ( string $code )
```
把字符串 code 作为PHP代码执行。 
###参数
1.  code
	需要被执行的字符串 

##base64_decode
(PHP 4, PHP 5)
base64_decode ― 对使用 MIME base64 编码的数据进行解码
###说明
```
string base64_decode ( string $data [, bool $strict = false ] )
```
对 base64 编码的 data 进行解码。
###参数
1. data
	 编码过的数据。
2. strict
	 如果输入的数据超出了 base64 字母表，则返回 FALSE。
